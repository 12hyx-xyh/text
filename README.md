# 单个手写数字识别系统
基于MATLAB传统图像处理与归一化互相关模板匹配的手写数字OCR识别
本项目为《机器视觉与图像处理》课程大作业，实现单个手写数字（0~9）的自动识别。系统基于传统图像处理流水线与归一化互相关模板匹配算法，无需深度学习框架，在普通笔记本电脑上即可流畅运行。
# 核心功能
图像预处理：灰度化 → 高斯降噪 → Otsu自适应二值化 → 形态学开闭运算
数字定位：连通域分析筛选数字主体，剔除噪点干扰
尺寸归一化：裁剪数字 → 缩放至24×24 → 居中放置于28×28画布
模板匹配：归一化互相关遍历0~9模板
可信度判断：相关系数 < 0.3 时发出低可信度警告
全流程可视化：每一步处理结果独立弹窗展示
